Con Wp Advertising Management non dovrai più preoccuparti della gestione degli spazi pubblicitari all'interno del tuo sito web.
Dimenticati le affiliazioni con partner esterni sarai tu a decidere tipologia, durata e prezzo degli spazi in vendita sul tuo sito web.


== Description ==

Wp Advertising Management gestisce per te la pubblicità nel tuo sito web. 
Dalla vendita alla pubblicazione sarà tutto automatizzato dandoti la possibilità di incrementare i tuoi profitti bypassando affiliazioni o prodotti del settore.


## Features

--VERSIONE GRATUITA--

*Tipologie di spazi pubblicitari disponibili: Link o banner sotto ai post e sidebar banner
*Pubblicazione automatica degli annunci acquistati dai tuoi clienti per la durata corretta
*Supporto via faq e forum

--VERSIONE PREMIUM--

*Tutte le funzionalità della versione GRATUITA
*Tipologie aggiuntive di spazi pubblicitari: banner nella parte iniziale dei post e guest post
*Possibilità di creare annunci illimitati in base alla durata
*Annunci multipli con visualizzazioni random (aumentare la disponibilità di uno spazio pubblicitario)
*Report dei click per cliente (tranne guest post)
*Supporto prioritario via ticket


== Installation ==

Per installare il plugin via piattaforma Wordpress:
1. Accedi al pannello di amministrazione del tuo sito web
2. Vai alla sezione plugin
3. Seleziona aggiungi nuovo
4. ricerca wp advertising management

Per installare il plugin con la procedura manuale:
1. Carica 'wp_advertising_management' nella cartella 'wp-content/plugins/'
2. Attivalo tramite la sezione Plugins

To install the plugin, follow the steps below

1. Upload `wp_management_advertising` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
